### FEATURES
- Insert Data
  - isi data NIM, Nama, Nilai, dan jalurSeleksi
  - klik ```add``` untuk menambahkan data
  - klik ```cancel```jika tidak jadi input data. Karena cancel, maka field form akan kosong karena otomatis reset
